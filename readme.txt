== Installation ==

Since this plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed on your website.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen in WordPress.
2.  Search for `Smart Addons For Elementor`.
3.  Install and activate the plugin, that's it.

= Manual Installation =
1.  Download [Smart Addons for Elementor](https://downloads.wordpress.org/plugin/smart-addons-for-elementor.zip "Download Smart Addons for Elementor").
2.  Extract the `smart-addons-for-elementor.zip` file. You'll get plugin files inside `smart-addons-for-elementor` directory.
3.  Upload the plugin directory to the `/wp-content/plugins/` directory.
4.  Activate the plugin through the 'Plugins' screen in WordPress.
